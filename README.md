Lights. Camera. Data | Power BI

This Power BI dashboard offers an interactive analysis of the top 50 movies spanning the 20th and 21st century.
It presents visual insights on movie ratings, genres, release trends, and performance across decades, making it easy to explore patterns in cinema history.
Built to practice data visualization and storytelling through Power BI, this project reflects skills in business intelligence and analytical thinking.
Developed as part of an academic portfolio for BCA 2nd Year at LJ University.
